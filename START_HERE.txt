⚡ INÍCIO RÁPIDO (5 MINUTOS)
================================================================================

PASSO 1: Crie o arquivo .env
------------------------------
1. Crie um arquivo chamado ".env" (sem aspas) nesta pasta
2. Cole este conteúdo dentro dele:

OPENAI_API_KEY=sua-chave-aqui

3. Substitua "sua-chave-aqui" pela sua chave da OpenAI
   (Obtenha em: https://platform.openai.com/api-keys)


PASSO 2: Execute o app
-----------------------
1. Abra o terminal/PowerShell NESTA PASTA
2. Ative o ambiente virtual:
   
   venv\Scripts\activate
   
3. Execute o app SaaS:
   
   streamlit run app_saas.py


PASSO 3: Acesse no navegador
-----------------------------
http://localhost:8501


PASSO 4: Crie sua conta
------------------------
1. Clique em "Criar Conta"
2. Preencha seus dados
3. Você ganha 3 análises GRÁTIS!


PASSO 5: Faça sua primeira análise
-----------------------------------
1. Vá em "Nova Análise"
2. Cole este endereço de teste:
   
   0x1f9840a85d5aF5bf1D1762F925BDADdC4201F984
   
3. Clique em "Analisar Contrato"
4. Aguarde 10-30 segundos
5. Veja o Score de Risco e análise completa!

🔑 APIs OPCIONAIS (Melhoram as Análises)
================================================================================

Sem estas chaves, o sistema funciona com dados SIMULADOS:

→ Obtenha em: https://etherscan.io/myapikey

NEWSAPI_KEY=  (notícias reais de cripto)
→ Obtenha em: https://newsapi.org/register

STRIPE_SECRET_KEY=  (pagamentos reais)
→ Obtenha em: https://dashboard.stripe.com/apikeys

Adicione no arquivo .env se quiser usar.

🐛 PROBLEMAS COMUNS
================================================================================

→ pip install -r requirements.txt

ERRO: "OpenAI API key not found"
→ Verifique se criou o arquivo .env com a chave

ERRO: Porta 8501 já em uso
→ streamlit run app_saas.py --server.port 8502

Análises muito lentas?
→ Use gpt-3.5-turbo em vez de gpt-4-turbo

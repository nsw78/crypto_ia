# ⚡ Início Rápido - 5 Minutos

## 🚀 Como Rodar Agora

### 1. Ative o ambiente virtual

```bash
# Você já tem o venv criado!
venv\Scripts\activate
```

### 2. Crie o arquivo .env

Crie um arquivo chamado `.env` na pasta `CRYPTO_IA_PROJECT` com este conteúdo:

```
OPENAI_API_KEY=sua-chave-aqui
```

**🔑 Onde conseguir a chave OpenAI:**
- Acesse: https://platform.openai.com/api-keys
- Faça login ou crie uma conta
- Clique em "Create new secret key"
- Copie e cole no arquivo .env

### 3. Instale as dependências (se ainda não fez)

```bash
pip install -r requirements.txt
```

### 4. Execute!

```bash
# Versão SaaS completa (RECOMENDADO)
streamlit run app_saas.py

# OU Landing Page
streamlit run landing_page.py

# OU Versão simples original
streamlit run app.py
```

### 5. Acesse no navegador

```
http://localhost:8501
```

## 🎯 Primeira Análise

1. **Crie uma conta** (ganha 3 análises grátis)
2. **Vá em "Nova Análise"**
3. **Cole um endereço de contrato**, por exemplo:
   ```
   0x1f9840a85d5aF5bf1D1762F925BDADdC4201F984
   ```
4. **Clique em "Analisar"**
5. **Aguarde 10-30 segundos**
6. **Veja o Score de Risco e análise completa!**
